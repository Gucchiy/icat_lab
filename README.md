# このリポジトリの説明
このリポジトリは、[立教大学](http://www.rikkyo.ac.jp/)[大学院ビジネスデザイン研究科 (立教ビジネススクール)](http://www.rikkyo.ac.jp/sindaigakuin/bizsite/)、 [深見嘉明研究室](http://icat-lab.tumblr.com/)の活動における情報共有ならびに情報発信のために開設されています。
